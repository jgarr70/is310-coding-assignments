## Lip Product Reviews

This program allows users to add and view comments about lip products.

### Features
- Add a comment about a lip product
- View previous comments left by other users
- Give each product a rating from **1 to 5**

### Rating Guidelines
When leaving a review, please include a rating using the following scale:

- **1** – Very poor  
- **2** – Poor  
- **3** – Average  
- **4** – Good  
- **5** – Excellent

Please make sure every review includes a rating between **1 and 5**.
