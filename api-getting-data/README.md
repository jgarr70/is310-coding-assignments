## why this API?
I chose the Spotify API, because I thought it would really cool to see my personal data and see what my listening stats are. Spotify does weekly stats, but I feel like they're a little off, so it was cool to see where it gets its data from. 

I got the API from their developer website and used the Spotipy library to extract the information I want. 

I chose the Beatles information from the Europeana Collection as they're literally music.
